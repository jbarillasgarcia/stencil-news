# Stencil News
Web component example using Stencil.

## Environment Configuration
1. Stencil 1.12.2
2. NPM 6.9.0
3. Visual Studio Code 1.40.0

## Custom Component
* search-query

## Clone & Run
```sh
> git clone https://github.com/jbarillasgarcia/stencil-news.git
> cd stencil-news/search-web
> npm install
> npm start
```
